# 옵티파이(Optify) 사업 컨텍스트 문서
> 용도: 공동 작업자(동생)가 옵티파이 사업 구조와 콘텐츠 운영 방식을 이해하기 위한 참조 문서
> 작성 기준일: 2026년 7월
> 이 문서의 내용은 내부 참조용이며, 콘텐츠에 그대로 복사해 쓰지 않는다.

---

## 1. 옵티파이는 어떤 회사인가

- 회사명: 옵티파이 (Optify) / optify.kr
- 대표: 박유리 (SEO·GEO 컨설턴트, "옵티파이 대표 박유리" 명의로 강의·유튜브·블로그 등 모든 대외 활동 통합)
- 소재지: 부산 (부산·경남이 거점이지만 전국 대상 사업)
- 업종: B2B 웹 에이전시 + SEO/GEO 컨설팅

### 핵심 포지셔닝
"검색과 AI에 발견되는 웹사이트 — 디자인이 아니라 구조에서 시작합니다."

옵티파이는 단순 홈페이지 제작사가 아니라, 전문직 사업자의 웹사이트와 콘텐츠를
구글 검색·AI 검색·네이버에서 발견되는 신뢰 자산으로 만드는 실행 파트너다.

### 핵심 서비스 4축
1. 워드프레스 기반 SEO 웹사이트 제작
2. GEO 콘텐츠 기획·발행 (구글·AI 검색 대상)
3. 네이버 검색·플레이스 구조 최적화 (광고 아님)
4. 유지보수 및 검색 노출 관리

### 타깃 고객
- 병의원, 치과, 한의원
- 법률사무소, 세무·회계사무소
- 학원, 부동산·로컬 비즈니스
- 검색 노출과 AI 노출이 필요한 전문직·서비스업 사업자

### 고객의 전형적인 문제
- 홈페이지는 있지만 구글 검색에 안 나온다
- 네이버 광고에만 의존하고, 광고를 끊으면 문의도 끊긴다
- AI(챗GPT 등)에 물어보면 경쟁사만 나온다
- 블로그 글은 있지만 홈페이지 자산으로 연결되지 않는다

## 2. 브랜드 체계 — 옵티파이와 리디웹의 구분

| 구분 | 옵티파이 | 리디웹 |
|---|---|---|
| 대상 | 사업자 (B2B) | 개인 학습자 |
| 성격 | 제작·컨설팅·운영 대행 | 워드프레스 수익화 교육 |
| 톤 | 전문적·신뢰형 | 친근한 교육형 |
| 콘텐츠 기준 | "이건 맡겨야겠다"는 생각이 드는 글 | 개인이 직접 따라 하는 글 |
| 애드센스·제휴마케팅 | 다루지 않음 | 다룸 |

- 리디웹은 개인 교육 브랜드로 유지하며, 장기적으로 동생에게 운영 이관 예정 (리디웹 카페 포함)
- 옵티파이 콘텐츠에서 리디웹은 신뢰 근거로만 가볍게 언급, 크게 노출하지 않는다
- 두 브랜드의 콘텐츠·디자인 톤이 섞이지 않도록 주의

---

## 3. 네이버 운영 방향 (동생 담당 영역)

### 기본 원칙: "광고가 아니라 구조로"
- 옵티파이는 네이버 광고 대행을 하지 않는다
- 네이버 검색·플레이스의 "구조 최적화"만 다룬다 (플레이스 세팅, 블로그 운영)
- 이 구분이 옵티파이의 "광고 의존 탈피" 메시지와 충돌하지 않는 핵심 논리다
- 콘텐츠에서 네이버를 깎아내리지 않는다. 네이버와 구글은 다르게 작동하며 둘 다 잡아야 한다는 관점 유지

### 네이버 계정
- 리디웹 계정을 재사용하지 않고, 옵티파이 전용 신규 계정 운영
- 페르소나: "박유리 (옵티파이 대표)" — 유튜브 페르소나 "SEO·GEO 박유리"와 정렬

### 네이버 블로그 카테고리 체계
| 카테고리 | 역할 |
|---|---|
| 옵티파이 안내 | 회사·서비스·진단 안내 (대표 직접 작성 — 담당자/GPT 작성 금지) |
| 사업자 홈페이지 제작 | 홈페이지 제작·비용·구조 관련 |
| 구글, AI 검색 | 구글 SEO·GEO·AEO·AI 노출 개념과 원리 |
| 네이버 검색, 플레이스 | 네이버 채널 관련 + 채널 통합 관점 |
| 업종별 마케팅 | 업종 특화 글 (현재 보류 — 반드시 대표 승인 후 작성) |
| 마케팅 진단, 가이드 | 점검법·선택 기준·체크리스트 |

### 네이버 블로그 1차 콘텐츠 플랜 (10편, 위에서부터 발행 우선순위)
| # | 제목(안) | 메인 키워드 | 카테고리 | 단계 | 상태 |
|---|---|---|---|---|---|
| N10 | 검색 노출 확인 방법, 네이버·구글·AI 한 번에 점검하기 | 검색 노출 확인 | 마케팅 진단, 가이드 | BOFU | 발행 (표준 예시) |
| N5 | 네이버 플레이스·블로그·홈페이지, 따로 맡기면 안 되는 이유 | 네이버 플레이스 관리 | 네이버 검색, 플레이스 | BOFU | 대기 |
| N1 | GEO 마케팅이란? SEO와 뭐가 다른지 5분 정리 | GEO 마케팅 | 구글, AI 검색 | TOFU | 대기 |
| N2 | 챗GPT에 물어봤을 때 우리 병원·사무소가 안 나오는 이유 | 챗GPT 노출 | 구글, AI 검색 | TOFU | 대기 |
| N4 | GEO 업체 고를 때 반드시 확인해야 할 5가지 | GEO 업체 | 마케팅 진단, 가이드 | BOFU | 대기 |
| N3 | SEO 홈페이지 제작 비용, 어디까지가 적정선일까 | SEO 홈페이지 제작 비용 | 사업자 홈페이지 제작 | BOFU | 대기 |
| N7 | 네이버 상위노출과 구글 SEO는 완전히 다릅니다 | 네이버 상위노출 | 네이버 검색, 플레이스 | TOFU | 대기 |
| N6 | 홈페이지 만들었는데 구글에 안 나와요 — 원인 7가지 | 홈페이지 검색 노출 | 사업자 홈페이지 제작 | MOFU | 대기 |
| N9 | GEO 홈페이지 제작 — 일반 홈페이지 제작과 다른 점 | GEO 홈페이지 제작 | 사업자 홈페이지 제작 | BOFU | 대기 |
| N8 | AEO 마케팅이란? AI 답변에 인용되는 원리 | AEO 마케팅 | 구글, AI 검색 | TOFU | 대기 |

- 발행 페이스: 주 1편 기준
- 발행 후 상태를 갱신한다 (대기 → 발행 + 발행일)

### 글별 각도 메모
- N5: 채널을 쪼개 맡기면 메시지가 어긋나 검색 신뢰가 깎인다는 관점. 특정 대행사 비판 금지, "구조" 문제로만 서술
- N1: GEO(생성형 엔진 최적화) 개념을 사업자 눈높이로. "SEO가 기반, GEO는 그 위에 얹는 것"이라는 순서 준수
- N2: 독자가 직접 챗GPT에 물어보게 유도 → 안 나오는 이유 설명 → N10 점검법으로 링크
- N4: "추천"이 아니라 "선택 기준" 각도. 타 업체 저격 금지
- N3: 저가·고가 양쪽의 함정 설명. "싸다/비싸다"가 아니라 "무엇이 포함되는가"로 판단하게
- N7: 네이버를 깎아내리지 않는다. 두 채널 모두 필요하다는 결론
- N6: 인덱싱·구조 문제를 쉬운 말로. site: 검색은 짧게 언급하고 N10 링크로 처리
- N9: "디자인이 아니라 구조에서 시작합니다" 브랜드 카피와 연결
- N8: N1(GEO)과 겹치지 않게 "AI 답변에 인용되는 원리" 중심

### 내부 링크 규칙
- 모든 글에서 자연스러운 위치에 N10(검색 노출 확인 방법) 링크 1회 삽입
- 같은 카테고리 내 발행된 글이 있으면 1개 이상 링크
- 발행 글 URL 목록에 없는 URL은 절대 만들어 쓰지 않는다 (가짜 URL 절대 금지)

## 4. 콘텐츠 작성 공통 원칙 (모든 채널)

### 톤
- 차분하고 신뢰감 있는 전문가 톤. 격식체 기본("~입니다") + 자연스러운 구어체 혼용 가능
- 전문 용어는 첫 등장 시 괄호로 쉬운 설명 병기. 예: GEO(생성형 엔진 최적화)
- 이모지·느낌표·클릭베이트 톤 금지
- 도입부 인사("안녕하세요, 옵티파이입니다") 금지 — 독자의 고민을 짚는 문장으로 바로 시작

### 절대 사용 금지 표현
- "100% 상위노출 보장", "검색 노출을 보장합니다"
- "AI가 무조건 추천하게 만듭니다", "ChatGPT에 반드시 노출됩니다"
- "검색의 시대는 끝났습니다"
- "단 N일 만에", "꿀팁", "비밀", "충격"
- "무조건", "보장", "반드시" 등 단정적 표현

### 권장 표현
- "검색과 AI가 이해할 수 있는 구조"
- "발견 가능성을 높이는 콘텐츠 설계"
- "광고 의존도를 낮추는 장기 콘텐츠 전략"
- "데이터에 따르면", "~할 가능성이 높습니다"

### 신뢰성 규칙
- 근거 없는 수치·통계를 만들어내지 않는다
- 가짜 URL 절대 금지. 출처가 확실하지 않으면 "관련 업계 보고서에 따르면"으로 표현하고 URL 생략
- 경쟁사·특정 업체 비방 금지
- 진단 관련 표현은 "검색 노출 점검"(SEO 중심)으로 통일 — "AI 노출 진단"이라는 표현은 쓰지 않는다

## 5. 의료광고법 컴플라이언스 (가장 중요)

병의원·치과·한의원 등 의료기관이 언급되는 콘텐츠는 반드시 대표(박유리)의 검수를 거친 후 발행한다.
담당자·GPT 단독 판단으로 발행 금지.

기본 금지 사항:
- 치료 전후 사진(before/after) 언급·게재
- 치료 효과·결과 보장 표현
- 경력·자격의 과장 (예: 자격이 없는데 "전문의" 표기)
- "최고", "유일", "1등" 등 최상급 표현
- 자극적인 임상 사진

변호사법·세무사법 등 다른 전문직 광고 규정도 동일하게 주의한다.

## 6. 작업 워크플로우 (2단계 승인 원칙)

1단계: 콘텐츠 플랜 기준으로 다음 글 주제·키워드·각도를 제안한다
→ 대표(또는 담당 권한자)의 승인을 받는다
2단계: 승인 후에만 초안을 작성한다

- 승인 없이 초안으로 넘어가지 않는다
- 플랜에 없는 주제는 카테고리 체계에 맞는 키워드·각도를 새로 제안하되 반드시 승인 후 작성
- 발행 후: 발행일 기록, 상태 갱신, 내부 링크는 발행 후 수동으로 추가

## 7. 옵티파이 워드프레스 블로그 (참고 — 대표 직접 운영 영역)

네이버 블로그와 별개로, optify.kr 블로그는 구글·AI 검색 대상 콘텐츠를 발행한다.

### 클러스터 구조
- 치과·법률·세무 3개 업종 중심 hub-spoke 클러스터
- 업종별 랜딩 페이지(/industries/dental-seo/ 등)가 hub, 블로그 글이 spoke
- 모든 글에 3종 내부 링크: hub 링크 + 같은 클러스터 spoke 링크 + 진단 페이지 링크

### 글 구조 표준
- 도입부 200자 안에 두괄식 답변 + 통계 1개
- H2는 질문형 또는 결론형으로만 작성 (정보형 H2 금지)
- 글 길이 2,000~3,500자, 단락은 3~5문장
- 결론 뒤 FAQ 5~6개
- 진단 CTA 최소 2회 (본문 중간 + 하단)
- 태그 사용 안 함, 1글 1카테고리 원칙

### 옵티파이 브랜드 비주얼
- Brand Black #0A0F1A, Signal Green #00E87B
- Pretendard 폰트
- Signal Green은 흰 배경 위 텍스트 색으로 사용 금지 (장식·배경용만)

## 8. 보안·비공개 원칙

- 협업 프로젝트·클라이언트의 실명, 계약 조건, 내부 단가는 콘텐츠에 절대 언급하지 않는다
- 옵티파이 자체 조사 데이터(AI 모델 인용 분석 등)는 대외 공개 금지 — 강의·내부 진단용으로만 사용
- 파트너사와의 협업 구조·조건 관련 내용은 이 문서 범위 밖이며, 필요 시 대표에게 직접 확인한다

## 9. 연락·확인 채널

- 판단이 애매한 경우의 우선순위:
  1) 읽는 사업자가 신뢰감을 느끼는가
  2) 검색·AI가 이해하기 좋은 구조인가
  3) 옵티파이의 핵심 가치(자산화·신뢰·장기)와 맞는가
  → 셋이 충돌하면 1번 우선
- 의료기관 관련 글, 플랜 외 주제, 가격·서비스 범위 언급이 필요한 글은 무조건 대표 확인 후 진행
